회의 참가자: 강민수 김승엽 신주환 이지호 조유성 최필립

1. 데이터셋 제작 방법
자동화 할 것인가? Yes -> 크롤링을 통해 데이터셋 제작
영상의 '태그'는 브라우저에서 크롤링 불가능 -> 태그를 확인할 수 있는 확장프로그램을 이용하여 수작업

2. 데이터셋 수집
속성 목록
제목
태그
조회수
영상길이
날짜
댓글 수
좋아요 수
추가적인 속성 필요 시 크롤링과 수작업을 통해 보충 예정

3. 데이터셋 크기
11/20일 기준으로 6개월 전 까지의 영상의 정보를 수집하기로 결정
프로토타입 개발 후 테스트하여 정확도와 신뢰도가 떨어진다면 1년 전 까지의 데이터를 추가로 수집하기로 결정

4. 진행결과
6개월 전 까지의 데이터셋 수집완료
깃허브에 프로젝트 레포지터리 생성 후 크롤링 코드와 문서 등록 완료
https://github.com/58TANK/Creator-Contents-Recommendation-System

5. 평가방법 의논
유명 유튜버에게 추천 결과로 나온 컨텐츠를 바탕으로한 영상을 올려 줄 수 있냐고 부탁 -> 허가받음
